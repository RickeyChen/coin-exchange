# Ricky Coin Exchange

## WHAT RICKY COIN EXCHANGE CAN DO ?
基于Java开发的货币交易所|BTC交易所|ETH交易所|数字货币交易所|交易平台|撮合交易引擎。\
本项目基于SpringCloudAlibaba微服务开发，可用来搭建和二次开发数字货币交易所。
